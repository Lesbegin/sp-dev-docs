# DwpFiles Element (Solution)

 
  
 **Applies to:** SharePoint 2016 | SharePoint Foundation 2013 | SharePoint Online | SharePoint Server 2013
  
Specifies the Web Part files (.dwp or .webpart) to include in the solution.
  
```
<DwpFiles>
  <DwpFile
    FileName = "Text" 
    Location = "Text">
  </DwpFile>
  ...
</DwpFiles>
```
