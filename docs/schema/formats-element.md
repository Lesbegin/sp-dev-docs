# Formats Element

 **Last modified:** March 09, 2015 
  
 * **Applies to:** SharePoint 2016 | SharePoint Foundation 2013 | SharePoint Online | SharePoint Server 2013 * 
  
Contains column and row formatting used in a Datasheet view.
  
```
<Formats>
  <Format
    Name = "Text">
      <FormatDef
        Type = "Text"
        Value = "Text">
            </FormatDef>
      ...
  </Format>
  ...
</Formats>
```
