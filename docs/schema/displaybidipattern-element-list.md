----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

The **DisplayBidiPattern** element applies to
sites whose locale identifier supports bidirectional reading order,
including Arabic, Farsi, Hebrew, and Urdu languages.


------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

The following example defines a display pattern for a <span
class="keyword">Title</span> column whose reading order is
bidirectional.
