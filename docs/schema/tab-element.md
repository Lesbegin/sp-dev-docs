<td align="left"><p>**CssClass**</p></td>

<td align="left"><p>**Description**</p></td>

<td align="left"><p>**Id**</p></td>
<td align="left"><p>Required. A string that identifies the control, such as "Ribbon.CustomTabExample".</p></td>
</tr>
<tr class="odd">
<td align="left"><p>**Sequence**</p></td>
<td align="left"><p>Optional. An integer that specifies the order of placement among other tab controls.</p></td>
</tr>
<tr class="even">
<td align="left"><p>**Title**</p></td>
<td align="left"><p>Optional. The title to display on the tab.</p></td>
