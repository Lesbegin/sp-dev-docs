# Rows (dsQueryResponse)

 
  
 **Applies to:** SharePoint 2016 | SharePoint Foundation 2013 | SharePoint Online | SharePoint Server 2013
  
Contains a collection of **Row** elements that specify the data from the list. 
  
```
<Rows>
  <Row></Row>
  <Row></Row>
  ...
</Rows>
```
