can be used within a [Where](where-element-query.md)
element in a query.

Remember that Collaborative Application Markup Language (CAML) is
case-sensitive; note the lowercase "eq" in the <span
class="keyword">Geq</span> element name.


------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

The following example queries for cases where the <span
class="keyword">Expires</span> field value either equals an empty string
or is greater than or equal to the current date:

## Definition
XML 
<span class="copyCode" onclick="CopyCode(this)"

    <Query>
      <Where>
        <Or>
          <IsNull>
            <FieldRef Name="Expires" />
          </IsNull>
          <Geq>
            <FieldRef Name="Expires" />
            <Value Type="DateTime">
              <Today />
            </Value>
          </Geq>
        </Or>
      </Where>
      <OrderBy>
        <FieldRef Name="Modified" Ascending="FALSE" />
      </OrderBy>
    </Query>
